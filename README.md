# Meal App (clean)

This is a clean Flutter project prepared to work with TheMealDB API.

How to use:
1. Ensure Flutter SDK is installed.
2. Copy this folder into your machine or download the ZIP.
3. From project root run:
   flutter pub get
   flutter run

The project includes:
- lib/models
- lib/services
- lib/widgets
- lib/screens
- lib/main.dart
